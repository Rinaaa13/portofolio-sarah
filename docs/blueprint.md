# **App Name**: Desmarina Dev Portfolio

## Core Features:

- Dynamic Profile Display: Showcase personal biography, key skills, and essential contact information including phone, email, LinkedIn, and GitHub profiles.
- Interactive Project Showcase: Present diverse projects (front-end, data analysis, UI/UX) with clear titles, roles, detailed descriptions, and direct links to live demos or documentation.
- Educational Background Highlight: Detail academic history, including university, major, GPA, and relevant coursework from Sriwijaya University.
- AI Portfolio Insight Tool: An AI tool that generates concise summaries or highlights key aspects from a selected project description or the overall portfolio for quick recruiter insights.

## Style Guidelines:

- Primary color: A rich, professional blue (#1B55B2) that signifies trust, stability, and intelligence, complementing a background of high desaturation.
- Background color: A subtle light blue-grey (#F2F5F8) with the same underlying hue as the primary color, providing a clean and understated canvas.
- Accent color: A vibrant cyan-blue (#24BBE6) for interactive elements and highlights, adding a touch of innovation and contrast while maintaining harmony.
- Headlines and prominent text: 'Space Grotesk' (sans-serif) to convey a modern, technical, and forward-thinking aesthetic.
- Body text and detailed descriptions: 'Inter' (sans-serif) for exceptional readability, clarity, and a neutral, professional feel across all content.
- Employ a consistent set of clean, minimalist line icons for social links, contact information, and skill representations to enhance clarity and visual appeal.
- Implement a clean, sectioned single-page layout that is fully responsive, ensuring optimal viewing experience on both desktop and mobile devices.
- Incorporate subtle hover effects on project cards and smooth scrolling transitions for navigation, adding a touch of interactivity without being distracting.
